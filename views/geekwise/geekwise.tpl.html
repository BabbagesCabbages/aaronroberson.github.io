Development Environment

Throughout this course we will be developing the MyStore e-commerce application. I've created a boilerplate template and have added it to Github. If you don't have one already, you'll be creating an account and forking the boilerplate template so that you can commit your work to Github.

Install Git

Download and install Git, selecting the option to add the PATH to your system path:

Install SourceTree

Download and install SourceTree, a Git GUI client for Windows or Mac.

We will be referencing Angular and other third-party scripts and resources that are hosted on a CDN. We will also be making RESTful API calls and will require that our web application is served up in the browser. To facilitate this, we will be using Node.js and GRUNT to serve up a local instance of the application in the browser.

Install Node.js

Download and install Node.js on your system.

Install GRUNT

GRUNT and GRUNT plugins are installed and managed via npm, the Node.js package manager. Once Node.js has been installed, you can install GRUNT from the Windows Console or Mac Terminal (You may need to use sudo (for OSX, *nix, BSD etc) or run your command shell as Administrator for Windows). 

npm install -g grunt-cli

The command above will put the grunt command in your system path, allowing it to be run from any directory.

Install Bower

Like GRUNT, you can install Bower from the console or terminal using the following command

npm install -g bower
