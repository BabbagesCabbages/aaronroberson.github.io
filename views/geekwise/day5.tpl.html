Day 5 - Featured Products

Tonight's class will be more of a hands-on class. On your own, you will be adding a featured products section to the home page of the 'MyStore' app. During tonight's class you will be:

  • Creating a controller for the home state
  • Using the 'ProductService' to create a featured products array
  • Adding featured products to the home view

Setting up

Change directories to your local app directory and start the web server:

grunt

You can now open your browser and go to http://localhost:9001 to access the app.

Step 1 - Create the HomeController

  1. Create the home controller in the '/controllers' directory
  2. Inject the $scope and 'ProductService' dependencies into the controller
  3. Initialize a 'featuredProducts' array literal on the $scope
  4. Loop through the results of the getProducts() method using angular.forEach():
    • Write an if statement to see if the 'isFeatured' property is true.
    • When true, push the product to $scope.featuredProducts array
  5. Add the script reference to 'index.html' page
  6. Add the HomeController to the 'home' state in 'routes.js'

Step 2 - Copy the HTML and CSS assets

  1. Go to http://codepen.io/DavieB/pen/CFywH/ and copy the HTML to the 'home.html' page (after the jumbotron) in the '/views' directory
  2. Copy the CSS (everything including and after the .featured .full definition) to a new CSS file named 'featured-products.css' in the '/assets/css' directory
  3. Update index.html to include a link to the CSS file

Step 3 - Bind the featuredProducts to the view

  1. Reference the first element in the featuredProducts array (remembering that arrays are 0-based) for the '.featured .full' product
  2. Use ng-repeat to loop through the featuredProducts array and build the '.featured_small' product list.
  3. Use the built-in Angular filter limitTo to only display 3 items in the '.featured_small' list.
  4. Use the ng-if directive when looping over the 'featured_small' list to only display products with an $index greater than 1 (keep in mind, $index is 0-based)

The ng-repeat directive includes the following special properties:

  • $index
  • $first
  • $middle
  • $last
  • $event
  • $odd

Step 4 - Update the 'read more' links

  1. Use the UI-router sref directive to update the 'read more' links to point to the selected product detail page

Step 4 - Commit your code to github

In SourceTree (or the github client or CLI of your choice), commit and push your changes.
