Function scope

Unlike traditional programming languages such as C, JavaScript does NOT have block level scope. JavaScript has function scope, meaning that variables defined within a function body are not accessible outside of the function body:

var x = 10;
function hello() {
	var y = 5;
	console.log(x); // logs 10
}
console.log(y); // throws undefined exception

In the code above, the variable y which is defined within the function body is not accessible outside of the function.

Callbacks and closures

A closure in JavaScript provides a convenient way to share function scope between two ore more functions:


function hello() {
	var y = 5;
	return y;
}
function world(callback) {
	console.log(callback());
}
world(hello); // throws undefined exception

Immediately Invoked Function Expressions

In JavaScript, the default scope for all variables and functions is the global scope, which is the DOM window. When developing large applications, you should limit what is added to the global scope. To do that, you can create a function that immediately invokes itself, otherwise known as an IIFE:

Constructor Functions

In JavaScript you can define classes (or objects) much like you define a function:

function Vehicle(make, model, doorCount, color) {
    this.make = make;
    this.model = model;
    this.doors = doorCount;
    this.color = color;
}

var truck = new Vehicle('Toyota', 'Tacoma', 4, 'grey');

The keyword this inside the function refers to the current object. By assigning variables using this you are adding properties to the object that will exist for every instance of the object.

Learn more about the this keyword, Understanding the 'this' keyword in JavaScript

The new operator creates an instance of a user-defined object type or of one of the built-in object types that has a constructor function.

Learn more about the new operator on the Mozilla Developer Network 
var truck = new Vehicle('Toyota', 'Tacoma', 4, 'grey');
console.log(truck.model); // Outputs Tacoma
var car = new Vehicle('Honda', 'Accord', 2, 'white');
console.log(car.model); // Outputs Honda

Object literals

When creating objects in JavaScript, you can create an object-literal, for example:

var Vehicle = {
    make: 'Toyota',
    model: 'Tacoma',
    doors: 4,
    color: 'grey'
};
