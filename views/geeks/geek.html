Download and install SourceTree, a Git GUI client for Windows or Mac.

We will be referencing Angular and other third-party scripts and resources that are hosted on a CDN. We will also be making RESTful API calls and will require that our web application is served up in the browser. To facilitate this, we will be using Node.js and Express to serve up a local instance of the application in the browser.
